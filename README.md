# WaraqaOS

A simple, interactive Operating System simulator built from scratch in C++. This project simulates core OS environments, including a command-line interface (CLI) and customizable system settings.

## Features

* **Command Line Interface (CLI):** A custom interpreter built to process user commands dynamically.
*   **User Management:** Modular architecture utilizing a dedicated `User` struct layout to manage session profiles.
*   **System Settings:** Interactive settings module allowing users to view and modify simulator environments.
